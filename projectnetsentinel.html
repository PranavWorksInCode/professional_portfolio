NetSentinel

Real-time network traffic monitoring and security analysis tool

Overview

NetSentinel is a sophisticated, real-time network traffic monitoring and security analysis tool designed to provide users with granular visibility into their system's network activity. Built to bridge the gap between complex command-line network utilities and modern user interfaces, NetSentinel allows users to visualize their network footprint dynamically. The system leverages a robust Python backend to interface directly with the network stack, utilizing advanced libraries for raw packet capture and process-level connection tracking. This backend streams telemetry data via WebSockets to a high-performance React frontend.

Technical Highlights

Key capabilities include a live, streaming bandwidth monitor that tracks upload and download throughput with millisecond latency, a comprehensive active connection table that correlates network sockets to specific system processes (PIDs), and an integrated port scanner for self-security auditing. Furthermore, NetSentinel features an educational packet sniffer that intercepts and decodes raw TCP/IP frames, displaying header information to demystify data transmission protocols. This project demonstrates proficiency in full-stack development, system-level programming, real-time data visualization, and cybersecurity fundamentals.
