Bloxxs

A 3D reimagining of the classic block-stacking puzzle game

Overview

Bloxxs takes the timeless mechanics of Tetris and expands them into a three-dimensional challenge. Designed to test spatial awareness and quick thinking, players must maneuver falling 3D polyominos onto a grid board. The game features smooth C++ performance, real-time score tracking, and an escalating difficulty curve that keeps players engaged.

Technical Highlights

Built entirely in C++, Bloxxs demonstrates efficient memory management and game loop optimization. The rendering engine handles 3D perspective and collision detection in real-time without relying on heavy external game engines, showcasing pure programming fundamentals.

Tech Stack

C++ OpenGL Game Design Algorithms

Key Features

  • Fully 3D Gameplay Environment
  • Custom Collision Detection
  • High Score System
  • Optimized Rendering Loop
